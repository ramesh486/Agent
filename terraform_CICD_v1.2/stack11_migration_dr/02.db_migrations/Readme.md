# OCI Database Migration (DMS)

OCI Database Migration (DMS) migrates Oracle databases from on-premises or cloud deployments to OCI. The easy-to-use graphical user experience validates and manages migration workflows. DMS transparently uses Oracle Zero Downtime Migration (ZDM), Cloud Premigration Advisor Tool (CPAT), Oracle Data Pump, and Oracle GoldenGate services for secure, fault-tolerant, and incremental migrations.


## Prerequisite

* Create a Vault
* Create Databases 
* Create an Object Storage Bucket
* Create Registered Databases
