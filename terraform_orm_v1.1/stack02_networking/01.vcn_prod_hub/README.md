Terraform Prod HUB VCN for Delta Dental Oracle Cloud Infrastructure Region

The Terraform Prod HUB VCN for Oracle Cloud Infrastructure provides a reusable Terraform module that provisions a Prod HUB VCN on OCI.

It creates the following resources:

    A Prod HUB VCN with  CIDR blocks

    An  internet gateway and a route table

    A NAT gateway

    A service gateway

    Dynamic Routing Gateway attachment

    Route Table
    
    Security List

    Private Subnets: Prod Trust; Prod Untrust; Prod Management
    
    Public Subnets: Prod NLB
