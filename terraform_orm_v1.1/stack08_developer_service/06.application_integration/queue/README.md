# Overview 
This is a Terraform configuration that creates the `queue` service on Oracle Cloud Infrastructure. 

The Terraform code is used to create a Resource Manager stack, that creates the required resources and configures the application on the created resources.
In the context of Oracle Cloud Infrastructure (OCI), a queue refers to the Oracle Cloud Infrastructure Queue service, which is a fully managed message queue service provided by Oracle. It enables you to send, store, and receive messages between different components of your applications or microservices.

The OCI Queue service offers reliable message delivery, ensuring that messages are persisted and retained until they are explicitly acknowledged. It supports both point-to-point and publish-subscribe messaging patterns.
