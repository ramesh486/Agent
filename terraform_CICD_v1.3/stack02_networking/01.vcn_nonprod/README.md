# Terraform NON-PROD VCN for Oracle Cloud Infrastructure

# The Terraform NON-PROD VCN for Oracle Cloud Infrastructure provides a reusable Terraform module that provisions a NON-PROD VCN on OCI.

# It creates the following resources:

    # A NON-PROD VCN with  CIDR blocks

    # An  internet gateway and a route table

    # A NAT gateway

    # A service gateway

    # Dynamic Routing Gateway attachment

    # Route Table
    
    # Security List

    # Private Subnets: AppDev; DBDev; AppTst; DBTst
