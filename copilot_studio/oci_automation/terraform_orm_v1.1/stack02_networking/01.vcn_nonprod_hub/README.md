Terraform NonProd HUB VCN:

The Terraform NonProd HUB VCN for Oracle Cloud Infrastructure provides a reusable Terraform module that provisions a NonProd HUB VCN on OCI.

It creates the following resources:

    A NonProd HUB VCN with  CIDR blocks

    An  internet gateway and a route table

    A NAT gateway

    A service gateway

    Dynamic Routing Gateway attachment

    Route Table
    
    Security List

    Private Subnets: NonProd Trust; NonProd Untrust; NonProd Management
    
    Public Subnets: NonProd NLB
